<Tabs syncKey="os">
  <TabItem label="Linux">Du behöver [`Git`](https://git-scm.com/), [`CMake`](https://cmake.org/) och en `C++` kompilator installerad.   <TabItem label="macOS">Du behöver [`Git`](https://git-scm.com/), [`CMake`](https://cmake.org/) och en `C++` kompilator installerad.</TabItem>
	<TabItem label="Windows">Du behöver [`Git`](https://git-scm.com/), [`CMake`](https://cmake.org/) och en `C++` kompilator installerad.</TabItem>

## Install

```sh
git clone https://github.com/UnknownFreeOccupied/ufo.git
cd ufo
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```
