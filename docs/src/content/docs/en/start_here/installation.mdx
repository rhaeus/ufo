## Prerequisite

<Tabs syncKey="os">
  <TabItem label="Linux" icon="linux">You need to have [`Git`](https://git-scm.com/), [`CMake`](https://cmake.org/), and a `C++` compiler installed. To use parallel processing, you may need to install [`oneTBB`](https://uxlfoundation.github.io/oneTBB/) or [`OpenMP`](https://www.openmp.org/) as well.</TabItem>
  <TabItem label="macOS" icon="apple">You need to have [`Git`](https://git-scm.com/), [`CMake`](https://cmake.org/), and a `C++` compiler installed.</TabItem>
	<TabItem label="Windows" icon="seti:windows">You need to have [`Git`](https://git-scm.com/), [`CMake`](https://cmake.org/), and a `C++` compiler installed.</TabItem>
</Tabs>

## Install

```sh
git clone https://github.com/UnknownFreeOccupied/ufo.git
cd ufo
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```
